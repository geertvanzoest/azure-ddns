**Steps:**

1. `actions/checkout@v4` -- checkout de repo

2. Installeer jq via apt:
   ```
   sudo apt-get update && sudo apt-get install -y jq
   ```

3. Installeer bats-core via git clone (NIET via apt -- niet beschikbaar op Ubuntu):
   ```
   git clone --depth 1 https://github.com/bats-core/bats-core.git /tmp/bats-core
   sudo /tmp/bats-core/install.sh /usr/local
   ```

4. Draai de test suite:
   ```
   bats test/
   ```

**Job naam:** `test`

- Geen environment variables nodig -- tests gebruiken mock waarden uit test_helper.bash
