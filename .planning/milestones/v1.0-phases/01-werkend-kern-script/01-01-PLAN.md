Maak het bestand `ns4j` in de repo-root. Het script volgt deze exacte structuur:

**Header en safety:**
- Shebang: `#!/bin/bash` (per D-06, absoluut pad)
- `set -euo pipefail`
- `trap 'log "ERROR" "Onverwachte fout op regel ${LINENO}"' ERR`

**Constanten (readonly):**
```bash
readonly API_VERSION="2018-05-01"
readonly AZURE_SCOPE="https://management.azure.com/.default"
readonly IP_SERVICE="https://icanhazip.com"
readonly EXIT_OK=0
readonly EXIT_CONFIG=1
readonly EXIT_IP=2
readonly EXIT_AUTH=3
readonly EXIT_DNS=4
```

**Functie `log()` (per D-01, D-02, OPS-02):**
- Parameters: `level` en `message`
- Timestamp via `date '+%Y-%m-%d %H:%M:%S'`
- Output formaat: `[TIMESTAMP] LEVEL: bericht`
- Als `level == "ERROR"`: output naar stderr (`>&2`)
- Anders: output naar stdout

**Functie `validate_config()` (per D-03, CFG-01, CFG-02):**
- Declareer array `required_vars` met exact deze 7 namen (per CLAUDE.md, niet REQUIREMENTS.md -- zie Pitfall 7):
  `AZURE_TENANT_ID`, `AZURE_CLIENT_ID`, `AZURE_CLIENT_SECRET`, `AZURE_SUBSCRIPTION_ID`, `AZURE_RESOURCE_GROUP`, `DNS_ZONE_NAME`, `DNS_RECORD_NAME`
- Check ook `command -v jq` (jq is niet standaard op Pi)
- Loop over `required_vars`, voor elke ontbrekende: `log "ERROR" "${var} is niet ingesteld"`
- Teller bijhouden, als minstens 1 ontbreekt: `exit 1` (EXIT_CONFIG)
- Als jq niet gevonden: `log "ERROR" "jq is niet geinstalleerd (apt-get install jq)"` en `exit 1`

**Functie `get_access_token()` (per AUTH-01, AUTH-02):**
- POST naar `https://login.microsoftonline.com/${AZURE_TENANT_ID}/oauth2/v2.0/token`
- Gebruik `curl --silent --show-error --max-time 10 --write-out "\n%{http_code}"` pattern
- Body parameters via `--data-urlencode`:
  - `client_id=${AZURE_CLIENT_ID}`
  - `client_secret=${AZURE_CLIENT_SECRET}`
  - `scope=https://management.azure.com/.default`
  - `grant_type=client_credentials`
- Header: `Content-Type: application/x-www-form-urlencoded`
- Parse response: `http_code` via `tail -n1`, `body` via `sed '$d'`
- Check `http_code != 200`: log ERROR met body, `exit 3` (EXIT_AUTH)
- Extract token: `jq -r '.access_token // empty'` (per Pitfall 5, NOOIT zonder `// empty`)
- Check token leeg: log ERROR, `exit 3`
- Output token via `echo "$access_token"`

**Functie `get_public_ip()` (per IP-01):**
- GET `https://icanhazip.com` via `curl --silent --show-error --max-time 10 --write-out "\n%{http_code}"`
- Parse `http_code` en `body` (zelfde pattern als get_access_token)
- Check `http_code != 200`: log ERROR, `exit 2` (EXIT_IP)
- Trim whitespace van body: `ip=$(echo "$body" | tr -d '[:space:]')`
- Check IP leeg: log ERROR, `exit 2`
- Output IP via `echo "$ip"`

**Functie `get_dns_record()` (per DNS-01, D-04):**
- Parameter: `access_token`
- URL: `https://management.azure.com/subscriptions/${AZURE_SUBSCRIPTION_ID}/resourceGroups/${AZURE_RESOURCE_GROUP}/providers/Microsoft.Network/dnsZones/${DNS_ZONE_NAME}/A/${DNS_RECORD_NAME}?api-version=${API_VERSION}`
- GET met `Authorization: Bearer ${access_token}` header
- Gebruik `--write-out "\n%{http_code}"` pattern
- HTTP 200: extract IP via `jq -r '.properties.ARecords[0].ipv4Address // empty'`, output via echo
- HTTP 404: output lege string via echo (record bestaat niet, forceer update per D-04)
- Andere HTTP status: log ERROR met status en body, `exit 4` (EXIT_DNS)

**Functie `update_dns()` (per DNS-02):**
- Parameters: `access_token`, `new_ip`
- TTL: `local ttl="${DNS_TTL:-300}"` (optioneel, default 300)
- Bouw JSON payload via jq: `jq -n --arg ip "$new_ip" --argjson ttl "$ttl" '{properties: {TTL: $ttl, ARecords: [{ipv4Address: $ip}]}}'`
- PUT naar zelfde URL als get_dns_record (zonder query param verandering)
- Headers: `Authorization: Bearer ${access_token}` en `Content-Type: application/json`
- Gebruik `--write-out "\n%{http_code}"` pattern
- HTTP 200 of 201: log INFO "DNS record bijgewerkt: ${DNS_RECORD_NAME}.${DNS_ZONE_NAME} -> ${new_ip}"
- Andere status: log ERROR met status en body, `exit 4` (EXIT_DNS)

**Functie `main()` (orchestratie):**
```
1. validate_config
2. access_token=$(get_access_token)
3. public_ip=$(get_public_ip)
4. current_ip=$(get_dns_record "$access_token")
5. Als current_ip == public_ip:
     log "INFO" "IP ongewijzigd (${public_ip})"
     exit 0
6. Als current_ip leeg:
     log "INFO" "Geen bestaand A-record gevonden, wordt aangemaakt"
   Anders:
     log "INFO" "IP gewijzigd: ${current_ip} -> ${public_ip}"
7. update_dns "$access_token" "$public_ip"
8. exit 0
```

**Script afsluiting:**
```bash
main "$@"
```
